# 豪华庭院 (DST) - 开始说明

这是一个从零搭建的 DST 建筑 Mod 雏形，当前版本已包含：

- 可制作的房屋门（室外门）
- 进入室内分区（同一地图中的隐藏区域）
- 室内返回门（自动配对）
- 可摆放的装饰家具（盆栽）
- 家具网格吸附（默认 1 格）
- 支持多人进入同一建筑

## 安装方式

1. 将本目录作为 Mod 目录放入 DST 的 `mods` 文件夹。
2. 在世界创建或服务器配置中启用本 Mod。

## 当前配方

- `House Door`：木板 x4，石砖 x4
- `Potted Plant Kit`：草 x6，树枝 x3

## 已实现规则

- 室内家具只能在进入建筑后摆放（玩家具有室内标记时）。
- 摆放会自动吸附到网格点，保证整齐。
- 锤掉室外门会同时移除对应室内门。

## 配置项

- `Interior Size`：8/10/12（默认 10）
- `Furniture Grid Size`：1.0/0.5（默认 1.0）
